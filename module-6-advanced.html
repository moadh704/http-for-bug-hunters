MODULE 6

Advanced Techniques

Once the fundamentals are solid, these techniques separate average hunters from strong ones. They often require specific conditions but can have very high impact.

Learning Objectives

  • Understand advanced HTTP attacks and when they apply
  • Recognize the conditions required for these attacks
  • Know how to test for request smuggling, race conditions, and JWT issues

Key Advanced Topics

HTTP Request Smuggling

Caused by disagreement between front-end proxies and back-end servers about where one request ends and the next begins. Usually due to conflicting Content-Length and Transfer-Encoding headers.

Testing Focus: CL.TE and TE.CL desync attacks using tools like Burp's HTTP Request Smuggler extension.

Race Conditions

Occur when applications perform check-then-act operations without proper locking. Single-packet attack techniques have made these more reliable even over the internet.

Testing Focus: Turbo Intruder or similar tools for sending concurrent requests.

JWT Attacks

Common issues include algorithm confusion (alg: none), weak HMAC secrets, kid header manipulation, and JWT header injection.

Testing Focus: Use jwt_tool or similar to test different attack vectors.

GraphQL-Specific Issues

Introspection enabled in production, batching attacks to bypass rate limits, and deeply nested queries causing resource exhaustion.

Key Takeaways

  • These attacks require specific conditions but are often very high impact when successful.
  • Always check for proxy chains and load balancers when testing.
  • JWT issues remain very common in modern applications.
  • Advanced techniques often require specialized tools and patience.
