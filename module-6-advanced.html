Key Advanced Topics

HTTP Request Smuggling

Caused by disagreement between front-end and back-end servers on request boundaries.

Race Conditions

Check-then-act operations without proper locking. Single-packet attacks have made these more reliable.

JWT Attacks

Algorithm confusion, weak secrets, kid header manipulation.
