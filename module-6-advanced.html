MODULE 6

Advanced Techniques

Once fundamentals are solid, these techniques separate average hunters from strong ones.

Learning Objectives

  • Understand advanced HTTP attacks
  • Know when and how to apply them
  • Recognize the conditions required for these attacks to work

Key Advanced Topics

HTTP Request Smuggling

Caused by disagreement between front-end and back-end servers on where one request ends and another begins. Often due to conflicting Content-Length and Transfer-Encoding headers.

Race Conditions

When applications check-then-act without proper locking. Single-packet attacks have made these more reliable.

JWT Attacks

Algorithm confusion (alg: none), weak secrets, kid header manipulation, and more.

GraphQL Specific Issues

Introspection, batching attacks, and resource exhaustion through nested queries.

Key Takeaways

  • These attacks require specific conditions but can be very high impact.
  • Always check for proxy chains and load balancers.
  • JWT issues are still very common in modern applications.
