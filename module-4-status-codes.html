MODULE 4

Status Codes & Behavior

HTTP status codes are not just success/failure signals. They often act as powerful oracles during testing.

Learning Objectives

  • Understand what different status code classes reveal in practice
  • Learn how to use status codes as testing oracles
  • Identify information leakage through error responses

Status Code Classes & Security Relevance

2xx Success

Inconsistent responses between different privilege levels often reveal access control issues.

3xx Redirection

Open redirect vulnerabilities frequently live here.

4xx Client Errors

Differences in responses between valid and invalid users can confirm resource existence or bypass authorization.

5xx Server Errors

Goldmine for reconnaissance — stack traces, internal paths, and framework versions are often leaked.
