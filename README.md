# HTTP for Bug Hunters

> A high-quality, structured learning resource for understanding **HTTP** from a bug hunting and web security perspective.

This project delivers **clear theory + practical application** so bug bounty hunters, pentesters, and security researchers can deeply understand what happens on the wire.

**Goal**: Build one of the best free resources for learning HTTP internals with a security-first mindset.

---

## <img src="https://api.iconify.design/mdi/lightbulb-outline.svg?color=%2300d4c8" width="22" height="22" style="vertical-align: middle; margin-right: 8px;"/> Key Features

- **Structured Learning Path** — Progressive modules designed like a mini academy
- **Theory + Practice** — Every concept includes real-world testing approaches
- **Security Focused** — Built specifically for bug hunters and security professionals
- **High Quality Content** — Thoughtful, accurate explanations based on established practices
- **Clean Professional Design** — Modern dark UI across all pages
- **Practice Guidance** — Curated high-value lab recommendations (mainly PortSwigger)
